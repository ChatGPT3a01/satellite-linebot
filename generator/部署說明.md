# Netlify 部署說明

## 方法一：拖曳部署（最簡單）

1. 前往 https://app.netlify.com/
2. 登入或註冊帳號
3. 點擊「Add new site」→「Deploy manually」
4. 將整個 `generator` 資料夾拖曳到網頁上
5. 完成！會自動產生網址

## 方法二：GitHub 連結部署

### 步驟 1：上傳到 GitHub

```bash
# 初始化 Git
cd generator
git init
git add .
git commit -m "Initial commit"

# 建立 GitHub repo 後
git remote add origin https://github.com/你的帳號/satellite-linebot-generator.git
git push -u origin main
```

### 步驟 2：連結 Netlify

1. 前往 https://app.netlify.com/
2. 點擊「Add new site」→「Import an existing project」
3. 選擇「GitHub」
4. 授權並選擇你的 repo
5. 設定：
   - Branch: main
   - Build command: （留空）
   - Publish directory: （留空或 `.`）
6. 點擊「Deploy site」

## 自訂網域

部署完成後，可以設定自訂網域：

1. 在 Netlify 網站設定中
2. 點擊「Domain settings」
3. 點擊「Add custom domain」
4. 輸入你的網域
5. 依照指示設定 DNS

## 檔案結構

確保 `generator` 資料夾包含以下檔案：

```
generator/
├── index.html      # 主頁面
├── style.css       # 樣式
├── generator.js    # 生成邏輯
├── netlify.toml    # Netlify 設定
└── 部署說明.md     # 本說明文件
```

## 注意事項

- 這是純前端靜態網站，不需要後端
- 所有 API Key 都在使用者瀏覽器處理
- 不會儲存任何使用者輸入的資料
- 建議使用 HTTPS（Netlify 自動提供）
